implementations.problem_classes.LeakySuperconductor module¶

class LeakySuperconductor(Cv=1000.0, K=1000.0, u_thresh=0.01, u_max=0.02, Q_max=1.0, leak_range=(0.45, 0.55), leak_type='linear', order=2, stencil_type='center', bc='neumann-zero', nvars=128, newton_tol=1e-08, newton_iter=99, lintol=1e-08, liniter=99, direct_solver=True)[source]¶

Bases: ptype

This is a toy problem to emulate a magnet that has been cooled to temperatures where superconductivity is possible. However, there is a leak! Some point in the domain is constantly heated and when this has heated up its environment sufficiently, there will be a runaway effect heating up the entire magnet. This effect has actually lead to huge magnets being destroyed at CERN in the past and hence warrants investigation.

The model we use is a 1d heat equation with Neumann-zero boundary conditions, meaning this magnet is totally insulated from its environment except for the leak. We add a non-linear term that heats parts of the domain that exceed a certain temperature threshold as well as the leak itself.

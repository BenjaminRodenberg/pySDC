implementations.problem_classes.GrayScott_MPIFFT module¶

class grayscott_imex_diffusion(nvars=127, Du=1.0, Dv=0.01, A=0.09, B=0.086, spectral=None, L=2.0, comm=mpi4py.MPI.COMM_WORLD)[source]¶

Bases: ptype

Example implementing the Gray-Scott equation in 2-3D using mpi4py-fft for solving linear parts, IMEX time-stepping (implicit diffusion, explicit reaction)

mpi4py-fft: https://mpi4py-fft.readthedocs.io/en/latest/
